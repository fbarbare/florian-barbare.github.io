Personality Some info about me

My development skills mainly focus on JavaScript even though I have knowledge and can develop small applications using other programming languages.

As a JavaScript developer, I mainly worked using Vanilla JS. Because of my strong knowledge and understanding of Vanilla JS, I believe it brings me a huge advantage when getting into any new framework/library. I also learnt a lot about JavaScript patterns working in OOP and under MVC structures using frameworks such as RequireJS or Browserify.

I am currently trying to focus on learning more about AngularJS and Node.js as they are 2 of the emergent technologies in the world of web programming and like that I can manage any website/application with only one programming language.
